# StayNest
A full-stack web application inspired by modern rental platforms, built with custom UI design and complete frontend, backend, and database integration.
#Rental Platform Web App (Full Stack)
This is a full-stack rental platform web application built from scratch, 
inspired by how modern platforms like Airbnb function.

The project features a completely custom UI design, along with core 
functionalities such as user authentication, property listings, and booking system.

The goal of this project is to understand and implement real-world 
application architecture using frontend, backend, and database technologies.
